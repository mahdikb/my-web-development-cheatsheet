# 🧠 راهنمای ضد آلزایمر: چطور مثل یک حرفه‌ای کلاس بنویسیم و کلافه نشویم؟

اگر موقع نام‌گذاری کلاس‌ها مغزت قفل می‌کند، این فرمول ۳ مرحله‌ای و رازهای کدهای جوناس را بخوان.

---

## 🛑 راز اول: چرا کدهای جوناس را نمی‌فهمی؟ (فرمول لایه‌لایه)

جوناس وقتی یک کلاس می‌نویسد، از **کل به جزء** حرکت می‌کند. یعنی اسم کلاس را از چپ به راست می‌خواند:
`[اسم بخش بزرگ] - [اسم باکس کوچک‌تر] - [نقش خود المان]`

### 🔍 کالبدشکافی کلاسی مثل `chair-details` یا `testimonial-section`
* وقتی جوناس می‌نویسد `testimonial-section`:
  * بخش اول (`testimonial`): موضوع چیست؟ نظرات مشتریان.
  * بخش دوم (`section`): این المان چه حجم از صفحه‌ است؟ یک بخش بزرگ و کامل از سایت.
* وقتی جوناس می‌نویسد `chair-details`:
  * بخش اول (`chair`): موضوع چیه؟ صندلی.
  * بخش دوم (`details`): این المان چه نقشی دارد؟ جزییات و مشخصات فنی را نشان می‌دهد.

---

## 🛠️ فرمول ۳ مرحله‌ای برای ساختن کلاس خودت (بدون فکر کردن اضافی)

هر وقت خواستی برای یک تگ HTML کلاس بنویسی، این ۳ سوال را از خودت بپرس:

### ۱. من الان تو کدوم بخش (منطقه) سایت هستم؟ (Where)
* تو بخش هیرو هستم؟ ➔ `hero`
* تو بخش ویژگی‌ها هستم؟ ➔ `features`
* تو منوی بالام؟ ➔ `nav` یا `header`

### ۲. این المانی که دارم بهش کلاس می‌دم چیه؟ (What)
* یک باکسه؟ ➔ `box` یا `container` یا `card`
* یک تیتره؟ ➔ `title` یا `heading`
* یک عکسه؟ ➔ `img` یا `photo`
* یک متن معمولیه؟ ➔ `text` یا `desc` (مخفف description)

### ۳. حالا این دوتا کلمه رو با یک خط تیره (-) بچسبون به هم! (Combine)
* تیترِ بخش هیرو ➔ `hero-heading`
* عکسِ داخل کارت ویژگی‌ها ➔ `feature-img`
* دکمه‌ی بخش قیمت‌ها ➔ `pricing-btn`
* باکسِ نظرات مشتریان ➔ `testimonial-box`

به همین راحتی! دیگر نیازی نیست مچ دستت را روی پیشانی‌ات بگذاری و نیم ساعت فکر کنی.

## 🗂️ کاتالوگ کلمات پرکاربرد (این‌ها را حفظ کن)

برای اینکه کلمات کم نیاوری، این کاتالوگ استاندارد را همیشه کپی-پیست کن:

| کلمات مخصوص باکس‌های مادر (Containers) | کلمات مخصوص المان‌های داخلی (Content) |
| :--- | :--- |
| **`-container` / `-box`**: برای باکس‌های نگه‌دارنده بزرگ | **`-heading` / `-title`**: برای تمام تیترها (`h1` تا `h3`) |
| **`-wrapper`**: باکسی که یک یا دو المان را جهت فلکس‌باکس مچ کرده | **`-text` / `-desc`**: برای متن‌های معمولی پاراگراف (`p`) |
| **`-card`**: برای کارت‌های محصول یا خدمات | **`-link`**: برای تگ‌های لینک و ناوبری (`a`) |
| **`-list`**: برای باکس‌های تگ `<ul>` | **`-btn`**: مخفف button برای دکمه‌ها |
