# 📏 سیستم طراحی و خط‌کش اندازه‌دهی (Design System Rules)

جوناس در ابتدای استایل‌دهی، دو خط‌کش استاندارد (بر مبنای سیستم ۸ پیکسلی) بالای فایل قرار می‌دهد تا تمام اجزای سایت با هم هارمونی داشته باشند.

<details>
<summary><b>📐 ۱. خط‌کش فاصله‌ها (Spacing System)</b></summary>
<br>

* **لیست اعداد استاندارد (px):** `2 / 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 80 / 96 / 128`
* **قانون انتخاب:** تمام پدینگ‌ها، مارجین‌ها و `gap`های گرید و فلکس‌باکس در پروژه باید یکی از این اعداد (یا معادل rem آن‌ها) باشند.
* **ترفند سه گزینه‌ای جوناس:** برای پیدا کردن بهترین فاصله، ابتدا یک عدد کوچک و یک عدد بزرگ را از لیست فرضی انتخاب کرده و در نهایت روی عدد میانگین که بیشترین زیبایی بصری را دارد قفل می‌کند.

</details>

<details>
<summary><b>✍️ ۲. خط‌کش فونت‌ها (Font Size System)</b></summary>
<br>

* **لیست اعداد استاندارد (px):** `10 / 12 / 14 / 16 / 18 / 20 / 24 / 30 / 36 / 44 / 52 / 62 / 74 / 86 / 98`
* **قانون انتخاب:** برای اینکه متن‌ها سلسه‌مراتب داشته باشند، سایزها پله‌پله از این لیست انتخاب می‌شوند. مثلاً:
  * متن ریز کپشن: `14px` (`1.4rem`)
  * متن بدنه اصلی: `18px` (`1.8rem`)
  * عنوان کوچک: `24px` (`2.4rem`)
  * عنوان اصلی بخش: `44px` (`4.4rem`)
  * عنوان هیرو: `52px` (`5.2rem`)

</details>
<details>
<summary><b>👀 راهنمای تصویری: این سایزها دقیقاً کجای سایت اعمال می‌شوند؟</b></summary>
<br>

وقتی جوناس می‌گوید "عنوان هیرو" یا "کپشن"، منظورش بخش‌های مشخصی از ساختار صفحه است:

1. **👑 عنوان هیرو (Hero Heading - 52px / 5.2rem):**
   * **کجا استفاده می‌شود؟** اولین، بزرگ‌ترین و جذاب‌ترین تیتر سایت که در همان ثانیه اول در بالاترین بخش صفحه (Hero Section) می‌بینی.
   * **کد تگ:** معمولاً همان تک تگ `<h1>` صفحه است.

2. **🧱 عنوان اصلی بخش (Section Heading - 44px / 4.4rem):**
   * **کجا استفاده می‌شود؟** تیترهای اصلی شروع هر بخش جدید؛ مثلاً بالای بخش "ویژگی‌های ما"، "لیست قیمت‌ها" یا "فرم تماس با ما".
   * **کد تگ:** معمولاً تگ‌های `<h2>` سایت هستند.

3. **📦 عنوان فرعی / کوچک (Sub-heading - 24px / 2.4rem):**
   * **کجا استفاده می‌شود؟** تیترهای ریزی که داخل باکس‌ها یا کارت‌ها قرار می‌گیرند. مثلاً نام یک صندلی خاص یا عنوان یک کارت غذا.
   * **کد تگ:** معمولاً تگ‌های `<h3>` یا `<h4>` هستند.

4. **📝 متن بدنه اصلی (Body Text - 18px / 1.8rem):**
   * **کجا استفاده می‌شود؟** تمام پاراگراف‌ها، توضیحات طولانی، متن کارت‌ها و نوشته‌های معمولی سایت که کاربر باید آن‌ها را به راحتی بخواند.
   * **کد تگ:** تگ‌های `<p>`.

5. **🏷️ متن ریز یا کپشن (Caption / Subheading Label - 14px / 1.4rem):**
   * **کجا استفاده می‌شود؟** برچسب‌های خیلی کوچک و معمولاً رنگی که بالای تیترهای اصلی می‌آیند (مثلاً یک کلمه کوچک مثل "TESTIMONIALS" با حروف بزرگ بالای نظرات مشتریان) یا متن‌های خیلی ریز کپی‌رایت در فوتر سایت.
   * **کد تگ:** معمولاً تگ `<span>` با یک کلاس اختصاصی مثل `.subheading`.

</details>
