# ✍️ گزارش کامل و مو‌به‌مو: ویدیوی تئوری و پیاده‌سازی تایپوگرافی (جوناس شمدتمن)

این فایل شامل تمام جزئیات، اعداد استاندارد، ترفندها و خط به خط کدهایی است که جوناس در این دو ویدیو تدریس می‌کند.

---

<details>
<summary><b>🧠 ۱. بخش اول: تئوری و ۷ قانون طلایی تایپوگرافی (Web Design Rules)</b></summary>
<br>

جوناس در ویدیوی تئوری، موفقیت طراحی یک سایت را تا ۹۰٪ وابسته به تایپوگرافی می‌داند و این ۷ قانون را وضع می‌کند:

1. **انتخاب فونت (Font Choice):**
   * همیشه از فونت‌های استاندارد و بسیار خوانا استفاده کنید. فونت‌های خانواده **Sans-serif** (مثل Inter، Roboto یا Open Sans) برای متن‌های بدنه وب عالی هستند چون چشم را خسته نمی‌کنند.
   * هرگز بیش از ۲ فونت در سایت نیاورید (یکی برای عناوین، یکی برای متن‌ها).

2. **اندازه فونت (Font Size):**
   * سایز متن بدنه (Body Text) باید بین **16px تا 18px** (یا حتی 20px در سایت‌های مدرن) باشد تا کاربر مجبور به زوم نشود.

3. **وزن فونت (Font Weight):**
   * برای ایجاد تضاد، بین وزن متن معمولی و عناوین فاصله بگذارید. مثلاً متن معمولی با وزن `400` (Regular) و عناوین با وزن `600` (Semi-bold) یا `700` (Bold).

4. **ارتفاع خط (Line Height):**
   * فاصله بین سطرها در پاراگراف‌ها باید بین **1.5 تا 1.8** باشد. اگر کمتر از این باشد متن‌ها فشرده می‌شوند و اگر بیشتر باشد انسجام متن از دست می‌رود.
   * برای عناوین (Heading) چون سایز بزرگی دارند، `line-height` باید کمتر و بین **1.0 تا 1.2** باشد.

5. **طول خط (Line Length):**
   * طول ایده‌آل برای یک سطر متن، بین **45 تا 75 کاراکتر** است. خطوط خیلی طولانی خواندن را سخت می‌کنند. این کار با محدود کردن عرض باکس (`max-width`) انجام می‌شود.

6. **تضاد رنگی (Color Contrast):**
   * هرگز از مشکی مطلق (`#000000`) روی سفید مطلق استفاده نکنید! همیشه از خاکستری‌های بسیار تیره (مثل `#333` یا `#444`) استفاده کنید تا کنتراست ملایم‌تری ایجاد شود.

7. **سلسله‌مراتب بصری (Visual Hierarchy):**
   * مهم‌ترین اصل؛ کاربر باید در ۱ ثانیه اول بفهمد چه چیزی عنوان اصلی (`h1`)، چه چیزی عنوان بخش (`h2`) و چه چیزی متن ساده است. این کار با ترکیب تفکیک سایز، وزن و رنگ انجام می‌شود.

</details>

<details>
<summary><b>💻 ۲. بخش دوم: کدهای عملی و پیاده‌سازی تایپوگرافی در CSS</b></summary>
<br>

در ویدیوی پیاده‌سازی، جوناس این کدهای دقیق را برای پروژه می‌نویسد. توجه کنید که او از ترفند `62.5%` روی html استفاده می‌کند تا هر ۱ رم برابر ۱۰ پیکسل شود:

```css
/* ---------------------------------- */
/* 1) GLOBAL RESET & BASE SETTINGS    */
/* ---------------------------------- */
html {
  /* تبدیل سایز پایه مرورگر از 16px به 10px جهت محاسبات راحت با واحد rem */
  font-size: 62.5%; 
}

body {
  /* تعریف فونت اصلی، رنگ خاکستری تیره استاندارد و ارتفاع خط متون */
  font-family: "Inter", sans-serif;
  color: #343a40;
  line-height: 1; /* ریست اولیه خطوط */
}

/* ---------------------------------- */
/* 2) HEADINGS SYSTEM (سلسله مراتب عناوین) */
/* ---------------------------------- */

h1, h2, h3, h4, h5, h6 {
  color: #212529; /* رنگ تایرها کمی تیره تر و سنگین تر از متن معمولی است */
  font-weight: 700; /* تمام عناوین بولد هستند */
  letter-spacing: -0.5px; /* ترفند جوناس: چسباندن نامحسوس حروف در سایزهای بزرگ برای شیک شدن */
}

h1 {
  font-size: 5.2rem; /* معادل 52 پیکسل - مخصوص هیرو */
  line-height: 1.05; /* چون سایز بزرگ است، ارتفاع خط را کم میکنیم تا از هم نپاشد */
  margin-bottom: 3.2rem;
}

h2 {
  font-size: 4.4rem; /* معادل 44 پیکسل - مخصوص عنوان اصلی بخش ها */
  line-height: 1.2;
  margin-bottom: 4.8rem;
}

h3 {
  font-size: 3.0rem; /* معادل 30 پیکسل - مخصوص عناوین فرعی و داخل کارت ها */
  line-height: 1.3;
  margin-bottom: 1.6rem;
}

/* ---------------------------------- */
/* 3) PARAGRAPHS & TEXT ELEMENTS      */
/* ---------------------------------- */

p {
  font-size: 1.8rem; /* معادل 18 پیکسل - فوق العاده خوانا */
  color: #555555; /* رنگ کمی روشن تر از عناوین برای هدایت چشم کاربر */
  line-height: 1.6; /* فضای عالی برای نفس کشیدن پاراگراف */
  margin-bottom: 2.4rem;
}

/* استایل دهی به متون فرعی یا کپشن ها */
.subheading {
  font-size: 1.6rem;
  font-weight: 600;
  color: #e67e22; /* استفاده از یک رنگ تاکید (Accent) مثل نارنجی */
  text-transform: uppercase; /* تمام حروف بزرگ */
  letter-spacing: 0.75px; /* باز کردن فاصله حروف در متون تمام بزرگ */
  margin-bottom: 1.6rem;
}
